# azure-openai-webapp

A learning project for exploring OpenAI chat playground using default LLM (Large Language Model) and integrating it with a Python web application, then deploying it to Azure.

## Overview

This project demonstrates:
- Setting up and using OpenAI chat functionality with default LLM models
- Building a Python web application that integrates with OpenAI services
- Deploying the web application to Azure for cloud hosting

## Purpose

This repository serves as a hands-on learning experience for developers interested in:
- Understanding OpenAI API integration
- Building chat-based applications
- Deploying AI-powered web applications to Azure cloud platform

# Azure OpenAI Web App

A simple Flask web interface to chat with Azure OpenAI's GPT model.
